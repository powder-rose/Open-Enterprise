## Open Enterprise landing page

Это верстка одностраничного сайта с помощью HTML и CSS. Для позиционирования используется исключительно технология ***Grid*** 

## Инструкция
1. Склонируйте репозиторий с помощью команды <code>git clone</code>
```shell
git clone https://github.com/powder-rose/Open-Enterprise.git
```
2. Запустите при помощи live server в VS code или плагинов в Webstorm.
3. Готово!
